# Исследование данных и A/B-тестирование на примере бразильского e-commerce

## О проекте

Этот проект представляет собой комплексный анализ данных бразильского маркетплейса Olist с проведением A/B-теста в учебных целях. В ходе работы выполнены все этапы аналитического исследования: от предобработки данных до статистической проверки гипотез и визуализации результатов.

## Данные

Использован открытый датасет [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce). Данные содержат информацию о заказах, совершенных на маркетплейсе в 2016-2018 годах:

- **olist_orders_dataset** — информация о заказах (статус, даты)
- **olist_order_items_dataset** — детали товаров в заказах (цена, доставка)
- **olist_order_payments_dataset** — способы оплаты и суммы
- **olist_customers_dataset** — информация о клиентах
- **olist_products_dataset** — данные о товарах и категориях
- **olist_order_reviews_dataset** — отзывы покупателей

Общий объем данных: ~100 тысяч заказов.

## Структура проекта

### 1. Предобработка данных
- Приведение типов данных к корректным форматам
- Обработка пропусков
- Создание агрегированных признаков (средний чек, конверсия, количество товаров в заказе)

### 2. Разведочный анализ (EDA)
- Анализ распределения стоимости заказов (гистограмма, boxplot с логарифмической шкалой)
- Динамика конверсии по месяцам
- Распределение количества товаров в заказе
- Топ-10 категорий по выручке

### 3. Дизайн A/B-теста
- Формулировка гипотезы
- Определение тестового периода (август 2017)
- Рандомизация пользователей на контрольную и тестовую группы
- Проверка сопоставимости групп в предэкспериментальном периоде

### 4. Анализ результатов
- Сравнение средней стоимости заказа (AOV) между группами (t-test)
- Сравнение конверсии между группами (z-test для пропорций)
- Расчет размера эффекта (Cohen's d)
- Построение доверительных интервалов методом bootstrap
- Визуализация результатов (boxplot с логарифмической шкалой, barplot)
