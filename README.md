# Olist A/B Testing: Влияние ускоренной доставки на метрики e-commerce

## О проекте

Этот проект представляет собой полный цикл анализа A/B-теста на реальных данных бразильского маркетплейса Olist. Оценивается эффективность новой функции — обещания "Гарантированная доставка за 3 дня".

**Ключевой вопрос:** Влияет ли ускоренная доставка на средний чек (AOV) и возвращаемость пользователей?

## Данные

Использован открытый датасет [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce). Данные содержат информацию о 100 тысячах заказов, совершенных на маркетплейсе в 2016-2018 годах: детали заказов, платежи, доставка, отзывы и информация о клиентах.

## Использованные технологии

- **Python**: pandas, numpy (обработка данных)
- **Статистика**: scipy.stats (тест Манна-Уитни), bootstrap (доверительные интервалы)
- **Визуализация**: matplotlib, seaborn
- **Среда**: Jupyter Notebook
